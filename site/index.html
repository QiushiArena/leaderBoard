🏆 AI Detector Benchmark Leaderboard

Introduction

This is an open-source benchmark platform specifically designed to evaluate the performance of AI text detectors. We test texts across different domains and provide fair, transparent data references.

Metric Explanation

  • Score Mix: Weighted score across mix difficulty levels (0.2~0.8).
  • Score Iter: Weighted score across iteration levels (rewrite 4~16).
  • Score Rub: Robustness score based on performance stability.
  • Final Score: 0.5 * mix + 0.3 * iter + 0.2 * rub

Submission

If you would like your model to appear on the leaderboard, please submit a Pull Request on GitHub.
